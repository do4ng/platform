## run

1. Download Project

```bash
git clone https://github.com/kangdongmandoo/platform.git

cd platform
```

2. Install Modules

```bash
yarn # or npm install
```

3. set database
   [see more](./docs/run-project.md)

4. Run!

```bash
yarn dev # or npm run dev
```

[localhost:3000](http://localhost:3000)에서 호스팅됩니다.
